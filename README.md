# LiveTV

[Live Demo](https://livetv0076.netlify.app/)

A live TV streaming web application built with React.js and Tailwind CSS.

## Features

- Live TV streaming via HLS
- Channel categories (Sports, Music, News, Entertainment)
- Favorites with persistent storage
- Search channels
- Adaptive quality streaming
- Responsive design (mobile, tablet, desktop, TV)

## Tech Stack

- React.js
- Tailwind CSS
- Zustand (state management)
- HLS.js (stream playback)
- React Router DOM
- Framer Motion (animations)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

The app is configured for deployment on Vercel.
